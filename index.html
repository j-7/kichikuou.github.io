鬼畜王 on Chrome

このブラウザでは利用できません。

アリスソフトの 配布フリー宣言 に基づいて配布されている「鬼畜王ランス」を Google Chrome 上で遊べるようにしたものです。

インストール
ゲーム開始

詳細

アリスソフトのゲームエンジン System3.5 の有志によるフリー実装である xsystem35 を Portable Native Client に移植したものです。Windows, Mac, Linux, Chrome OS 上の Chrome で動きます。

ソースコードは以下の場所にあります。

  • https://github.com/kichikuou/xsystem35-nacl - xsystem35 の PNaCl 版
  • https://github.com/kichikuou/kichikuou.github.io - このサイト
