インストール手順

  1. アリスソフト アーカイブズ で配布されている鬼畜王ランスの CDイメージをダウンロードします。
  2. ダウンロードしたファイル KICHIKU_WIN.RAR は RAR 形式の圧縮ファイルなので、展開して CD-ROM イメージを取り出します。 Windows なら LhaPlus、 Mac なら The Unarchiver で展開できます。 ChromeOS は標準機能で RAR を展開できます。
  3. 展開したフォルダにある KICHIKUOU.img と KICHIKUOU.cue を、このウィンドウにドラッグ＆ドロップするか、下の「ファイルを選択」ボタンを押して選択します。
  4. ファイル保存の許可を求めるポップアップが表示されたら、「許可」をクリックします。（CD-ROM イメージから取り出したファイルを保存するために必要です。）約 650MB の空きディスク容量が必要です。
  5. インストールが始まります。インストール完了後は CD イメージを削除しても構いません。
